Path Finder

Find your path through A* algorithm.

API

End point : /path/a-star

params :
size : Integer
start : String (numbers concatinated by `,`)
end : String (numbers concatinated by `,`)
wall : String (numbers concatinated by `,` and `|`)
Example : /path/a-star?size=10&start=4,1&end=6,7&wall=4,4|5,4|6,4
